# customer-churn-prediction
## 📌 Problem Statement

Customer churn — when a subscriber stops using a product or service — costs businesses billions of dollars annually. Telecom companies lose roughly 1.9% of their subscriber base monthly. Acquiring a new customer costs 5–7× more than retaining an existing one. Yet most companies lack the predictive infrastructure to act before a customer leaves.

This project builds a machine learning system that:
- Ingests customer behavioral and demographic data
- Predicts the probability of churn for each customer
- Explains the key drivers behind each prediction
- Surfaces actionable insights through a dashboard

The goal is to enable retention teams to intervene proactively and reduce customer loss.
